# 🛡️ Detecção de Fraude com SQL

## 📌 Sobre
Projeto que simula regras de detecção de fraude em transações financeiras.

## 🚨 Regras aplicadas
- Transações acima de 10.000
- Transações em cidades diferentes no mesmo dia
- Alto volume de transações em curto período

## 🎯 Objetivo
Simular cenários reais de segurança em sistemas bancários.

## 🛠️ Tecnologias
- SQL Server
